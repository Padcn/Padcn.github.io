Http属于应用层打面向对象的协议
特点：

  1. 支持c/s模式
  2. 简单快速：客户向服务器请求服务时，只需请求方法和路径。
  3. 灵活：允许传输任意泪下打数据对象。以Content-Type加以标记区分
  4. 无连接：每次连接只处理一个请求，处理完客户请求，并收到用户应答即断开连接。可以节省传输时间。
  5. 无状态：是无状态协议，对事务处理没有记忆能力，如需要之前的信息则需要重传，导致每次连接的数据量增大。如不需要则应答比较快。
